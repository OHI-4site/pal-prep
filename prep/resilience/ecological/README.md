This folder contains sub-folders for all ecological resilience layers in the Palmyra Ocean Health Index assessment.

| Climate Change         | cc     |
| Ecosystem              | eco    |
| Fishing Pressure       | fp     |
| Habitat Destruction    | hd     |
| Invasive Species       | inv    |
| Pollution              | po     |
